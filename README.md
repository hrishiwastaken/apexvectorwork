# Therapy practice website mockups

Standalone, self-contained HTML mockups for a psychology / therapy practice.
Each file inlines its own fonts (Fraunces, Mulish, Caveat as data-URI woff2) and
illustrations, so it opens in any browser with no build step or network access.

## Files

- **understory.html** — "Understory", a warm hand-annotated editorial mockup:
  parchment + walnut-ink palette, soft-serif display type, hand-drawn wavy
  dividers, a slow breathing circle, ink-drawn marks, and margin doodles that
  point at the copy and the founder/team photos.
